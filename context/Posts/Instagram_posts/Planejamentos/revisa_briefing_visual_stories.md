# Revisa — Briefing Visual: Produção dos Stories

> Para usar no CapCut ou passar para quem vai produzir.

---

## Especificações técnicas

| Item | Definição |
|---|---|
| Formato | Vertical 9:16 |
| Resolução | 1080 x 1920px |
| Duração | 10–15 segundos por Story |
| Ferramenta sugerida | CapCut (grátis) |
| Exportação | MP4, sem faixa de corte nas bordas |
| Voz em off | ❌ Não usar — o texto carrega a mensagem sozinho |
| Áudio | ✅ Música instrumental de fundo, volume 15–20% |

## Identidade visual — cores oficiais da marca

| Nome | Hex | Uso nos Stories |
|---|---|---|
| Dark Navy | `#07191D` | Fundo principal de todos os Stories |
| Teal | `#32969B` | Label superior, palavras-chave em destaque, botão CTA |
| Off-White | `#FAFAFA` | Texto principal e corpo |

---

## Tipografia oficial

**Família:** Nunito — baixar grátis em fonts.google.com/specimen/Nunito

| Uso | Peso | Tamanho sugerido |
|---|---|---|
| Label superior (ex: "Para ex-funcionários") | Light 300, caixa alta | 18–20px |
| Título principal | ExtraBold 800 | 64–72px |
| Corpo / subtítulo | Regular 400 | 36–42px |

---

## Estrutura visual de cada Story

Inspirada na hierarquia do site — label + título grande + subtítulo curto + CTA.

```
┌─────────────────────────────┐
│  [logo]            [revisa] │  ← canto superior direito
│                             │
│                             │
│  LABEL EM TEAL PEQUENO      │  ← Nunito Light, caixa alta
│                             │
│  Título grande              │  ← Nunito ExtraBold
│  em Off-White com           │
│  palavra em Teal            │
│                             │
│  Subtítulo curto em         │  ← Nunito Regular, opacidade 70%
│  Off-White mais suave       │
│                             │
│                             │
│  [ BOTÃO CTA EM TEAL ]      │  ← fixo no rodapé
└─────────────────────────────┘
```

---

## Logo

- Versão: branca sobre fundo escuro
- Posição: canto superior direito
- Tamanho mínimo: 24px de altura
- Símbolo borboleta sempre com gradiente azul original

---

## Música de fundo

- Estilo: instrumental suave, "corporate lo-fi" ou "soft background music"
- Volume: 15–20% — quase imperceptível, só preenche o silêncio
- Onde encontrar no CapCut: aba Áudio → Músicas → Vibe → Corporate ou Calm
- Alternativa gratuita: pixabay.com/music (buscar "corporate background")
- Não usar música com letra — compete com a leitura do texto

---

## Animação do texto (CapCut)

- Texto aparece frase por frase — nunca tudo de uma vez na tela
- Transição sugerida: fade in (0.3s) ou corte direto
- Cada frase fica na tela por 2.5–3 segundos antes da próxima aparecer
- Não usar animações chamativas (bounce, zoom, glitch) — foge do tom da marca

- [ ] Fonte Nunito instalada no CapCut antes de começar

- [ ] Sem palavras: processar, ação, indenização, justiça, condenação
